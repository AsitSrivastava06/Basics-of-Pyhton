## ğŸ“– Theory (Tuples in Python)

A **tuple** is an ordered collection of elements enclosed in parentheses `()`.

### ğŸ”¹ Key Characteristics of Tuples
- **Ordered**: Elements maintain a fixed sequence.
- **Immutable**: Once created, elements cannot be added, removed, or changed.
- **Heterogeneous**: Tuples can store different data types together.
- **Allows duplicates**: Same value can appear multiple times.
- **Efficient**: Tuples are more memory-efficient and slightly faster than lists.

### ğŸ”¹ Why Tuples are Used
Tuples are preferred when:
- Data should remain constant and protected from modification  
- A fixed collection of values needs to be stored  
- Returning multiple values from a function  
- Using elements as dictionary keys (tuples are hashable)
